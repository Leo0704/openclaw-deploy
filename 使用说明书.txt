================================================================================
一、软件简介
================================================================================

龙虾助手是一款帮你一键部署和管理 OpenClaw 的工具软件。

简单来说，它能帮你：
  ✓ 激活软件（输入激活码）
  ✓ 一键安装 OpenClaw（不用自己敲命令）
  ✓ 配置 AI 模型（比如 Claude、GPT 等）
  ✓ 启动/停止 OpenClaw
  ✓ 安装各种技能插件
  ✓ 配置消息通知（Telegram/飞书）
  ✓ 更新软件

================================================================================
二、环境要求
================================================================================

【运行环境】
  下载的二进制版本已自带 Node.js 运行时，无需额外安装。

  如果需要运行源码版本：
  - Node.js >= 22.12.0
    下载地址: https://nodejs.org/（选 LTS 长期支持版）

【其他要求】
  - 网络能访问 GitHub 或国内镜像（用于下载离线包）
  - 硬盘至少预留 5GB 空间
  - 端口 18790 没被其他程序占用

【关于离线包模式】
  龙虾助手采用"离线包"模式部署 OpenClaw：
  - 无需 Git
  - 无需 pnpm/npm
  - 离线包已包含 Node.js 运行时和所有依赖
  - 部署更快、更稳定

三、下载安装
================================================================================

请根据自己的电脑选择对应的安装包：

--------------------------------------------------------------------------------
【Apple 电脑用户（M1/M2/M3/M4 芯片）】
--------------------------------------------------------------------------------

1. 先确认芯片类型：
   点击苹果图标 → 关于本机 → 看到"芯片"就是 M 系列

2. 下载: lobster-macos-arm64-app.zip

   适用电脑：
   - MacBook Pro (2021年及以后)
   - MacBook Air (2020年及以后)
   - Mac mini / Mac Studio / iMac (2021年及以后)

--------------------------------------------------------------------------------
【Windows 电脑用户】
--------------------------------------------------------------------------------

下载: lobster-win-x64.exe

适用于 Windows 10/11 64位系统

--------------------------------------------------------------------------------
【Linux 用户】
--------------------------------------------------------------------------------

下载: lobster-linux-x64

================================================================================
四、安装与启动
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
【Mac 电脑安装】
--------------------------------------------------------------------------------

1. 解压下载的 zip 文件
2. 得到 "Lobster Assistant.app"
3. 双击打开

如果提示"无法打开"：
  - 打开"系统设置" → "隐私与安全性"
  - 点"仍要打开"

或者在终端执行：
  xattr -cr "Lobster Assistant.app"

--------------------------------------------------------------------------------
【Windows 电脑安装】
--------------------------------------------------------------------------------

1. 双击下载的 exe 文件
2. 如果弹出安全警告，点"更多信息" → "仍要运行"

--------------------------------------------------------------------------------
【Linux 安装】
--------------------------------------------------------------------------------

1. 打开终端，进入下载目录
2. 执行: chmod +x lobster-linux-x64
3. 执行: ./lobster-linux-x64

--------------------------------------------------------------------------------
【启动成功后】
--------------------------------------------------------------------------------

浏览器会自动打开控制台页面

  地址: http://localhost:18790

如果这个端口被占用，会自动换成 18791、18792 等

================================================================================
五、激活授权
================================================================================

首次使用需要激活。

【激活步骤】
1. 打开控制台后，在页面输入激活码
2. 点击"激活"按钮
3. 搞定！激活码会和这台电脑绑定

【购买激活码】
  淘宝搜索"龙虾助手"或访问：
  https://m.tb.cn/h.iW33Qi7?tk=MPQHUv32tQo%20CZ193

【注意事项】
  - 一个激活码只能绑定一台电脑
  - 换电脑需要重新购买
  - 每次启动会自动验证授权

================================================================================
六、部署 OpenClaw
================================================================================

激活成功后，开始安装 OpenClaw。

【部署步骤】
1. 在控制台点击"部署"或进入"配置"页面
2. 选择模型服务商（自定义 API 或预设服务商）
3. 填写 API Key 和模型信息
4. 点击"开始部署"
5. 等待安装完成（约需 5-15 分钟）

【离线包模式说明】
  - 系统会自动检测离线包
  - 如果未找到，会提示你下载离线包
  - 下载"龙虾助手+离线包"压缩包后解压
  - 将离线包放在龙虾助手同一目录即可自动识别

【耐心等待】
  首次部署需要：
  - 验证/下载离线包
  - 解压安装包
  - 写入配置文件

  如果网络慢可能更久，请耐心等待，不要关闭程序

【部署失败怎么办】
  - 检查网络是否正常
  - 确认 Node.js 版本 >= 22.12.0
  - 试试用手机热点

七、配置 AI 模型（最重要的一步）
================================================================================

部署完成后，必须配置 AI 模型才能使用！

【什么是 API Key？】
  简单理解：相当于你的"会员卡号"
  - 用来识别你是谁
  - 用来扣费（按调用量收费）
  - 在模型服务商官网申请

--------------------------------------------------------------------------------
【方式一：选择已有的模型服务商】（推荐）
--------------------------------------------------------------------------------

支持的服务商和申请地址：

| 服务商 | 特点 | 申请地址 |
|--------|------|----------|
| Anthropic (Claude) | 推理能力强 | https://console.anthropic.com |
| OpenAI (GPT) | 综合能力强 | https://platform.openai.com |
| Google (Gemini) | 便宜大碗 | https://aistudio.google.com |
| OpenRouter | 聚合多模型 | https://openrouter.ai |
| 阿里云百炼 | 国内可直接用 | https://dashscope.console.aliyun.com |
| 阿里云 Coding | Code 模型 | https://coding.aliyun.com |
| DeepSeek | 性价比高 | https://platform.deepseek.com |
| 硅基流动 | 国内可用 | https://siliconflow.cn |
| Moonshot (Kimi) | 长文本处理强 | https://platform.moonshot.cn |
| 智谱 AI | 国内可用 | https://open.bigmodel.cn |

【配置步骤】
1. 进入"配置"页面
2. 选择你的服务商
3. 粘贴 API Key（从官网申请）
4. 选择要用的模型
5. 点"保存"

--------------------------------------------------------------------------------
【方式二：自定义 API】
--------------------------------------------------------------------------------

如果你用的不在上面的列表，按顺序填写：
1. Base URL - API 服务器地址（服务商会给）
2. API Key - 你的密钥
3. 兼容类型 - 选 OpenAI 或 Anthropic
4. Model ID - 模型名称
5. Endpoint ID（可选）- 用于标识此端点
6. 模型别名（可选）- 给模型起个简短名字
7. 点"测试连接"确认能用

【如何选择模型？】
  - 第一次用：选带"推荐"标签的
  - 想要便宜：选 Haiku、Flash 结尾的
  - 想要最强：选 Opus、Pro 结尾的

八、启动与使用 OpenClaw
================================================================================

【启动 OpenClaw】
1. 在控制台点击"启动"
2. 等待服务启动（约 10-30 秒）
3. 点"打开 OpenClaw"
4. 开始使用！

【停止 OpenClaw】
  点击"停止"按钮即可

【怎么用？】
  OpenClaw 是一个 AI 助手，你可以：
  - 和它聊天问问题
  - 让它帮你写代码
  - 分析文件/图片
  - 使用各种技能

十、消息通知配置
--------------------------------------------------------------------------------

配置后可以用Telegram/飞书和 OpenClaw 对话。

--------------------------------------------------------------------------------
【Telegram 配置】
--------------------------------------------------------------------------------

1. 打开 Telegram，找 @BotFather
2. 发送 /newbot 创建一个机器人
3. 记下机器人 Token（形如 123456:ABC-DEF1234ghIkl-zyx57W2vTcuAH外汇）
4. 在控制台粘贴 Token
5. 配置允许谁可以发消息

具体操作：
  - 在龙虾助手通知配置页面
  - 粘贴 Bot Token
  - 设置谁能用（私聊/群聊）

--------------------------------------------------------------------------------
【飞书配置】
--------------------------------------------------------------------------------

1. 打开 https://open.feishu.cn/
2. 创建企业应用
3. 获取 App ID 和 App Secret
4. 添加机器人权限：接收消息、发送消息
5. 获取 Verification Token
6. 在控制台填写这些信息

--------------------------------------------------------------------------------
【配置测试】
保存后可以点"测试连接"确认配置正确

十一、常见问题
--------------------------------------------------------------------------------

【Q: 打开后浏览器没反应？】
A: 自己打开浏览器访问 http://localhost:18790

【Q: 端口被占用？】
A: 会自动换端口，看终端输出的实际地址

【Q: 部署一直转圈圈？】
A: 首次部署需要较长时间，耐心等待
   如果超过 30 分钟没反应，关闭重试

【Q: 模型连接失败？】
A: 检查：
   1. API Key 复制对了吗？（别多空格）
   2. 那个服务商国内能访问吗？
   3. 账户里有钱吗？

【Q: 怎么更新软件？】
A: 启动时自动检查更新，也可以在设置里手动检查

【Q: 彻底卸载？】
A: 设置页面点"卸载 OpenClaw"，会删除所有相关文件

【Q: Mac 提示软件损坏？】
A: 终端执行：
   xattr -cr "/Applications/Lobster Assistant.app"

【Q: 提示需要下载离线包？】
A: 请下载"龙虾助手+离线包"压缩包，解压后放在龙虾助手同一目录
